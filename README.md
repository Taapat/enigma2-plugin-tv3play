TV3Play
=========
Enigma2 plugin to watch TV3 play online services

Depends
-------
python-json python-netclient python-twisted-web
 twisted need also python-compression

Download
-------
[enigma2-plugin-tv3play](http://taapat.ho.ua/enigma2-plugin-tv3play/)
